# iRes線上智能預約系統
本系統是針對診所與病患所開發的預約系統，期望整合小診所們線上預約服務，省去僅能傳統電話掛號、不確定人數、在現場久候的看診困境。
病患可以隨時線上預約，並看見即時叫號，收到即將到號通知再前往診所，節省等候時間，提升看診體驗，且可服務特殊群體(如：不方便撥電話的聾啞人士)。
診所透過數位化管理，亦可降低人力掛號負擔，增進效能，並可在網路上提升曝光度，促進病患前往看診意願。
